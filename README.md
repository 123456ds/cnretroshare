建议下载便携版，解压即用，或者安装包选着portable安装模式，即是便携版，方便备份（因为没有服务器，用户档案，好友列表存储在本地）。

------------


因为很多人用了最新版，官方版本在识别链接时会显示证书错误，所以分享个开发版，连接上人后，可以自己在RetroShare频道内升级（4.8/5.9是开发环境的区别，RetroShare软件的功能是一样的）
